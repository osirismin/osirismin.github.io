闲敲棋子落灯花

2023年9月3日
# 玩物丧志

最近一些比较热门的电影下映后上线流媒体，白嫖的快乐才是真快乐。陆陆续续看了几部，例如《消失的她》《巨齿鲨2：深渊》……

消失的她
6.4
2022 / 中国大陆 / 悬疑 犯罪 / 崔睿 刘翔 / 朱一龙 倪妮
天真善良白富美柔弱可欺，阴险毒辣凤凰男谋财害命。人物脸谱化，剧情充满套路。

观后感：天真善良白富美柔弱可欺，阴险毒辣凤凰男谋财害命。

在短视频平台一直有“几分钟”看完的版本，大致是也了解是怎么回事情，少了解谜的乐趣以后，我对这部片子的评价更低了。故事逻辑上的硬伤就不摘指了，人物脸谱化，剧情充满套路。

据说是改编自前苏联电影《为单身汉设下的陷阱》，我没看过。解谜的框架像《看不见的客人》「对不起，不配相提并论」，又像前些年的国产恐怖片的套路，总之，要不就是姊妹，要不就是闺蜜，「扮鬼」吓男主，然后寻找真相替受害女子复仇，结尾反转一如既往生硬，细想全是 Bug。

巨齿鲨2：深渊
5.4
2023 / 美国 中国大陆 / 动作 科幻 冒险 / 本·维特利 / 杰森·斯坦森 吴京
抛开设定、逻辑和剧情不谈，单说巨齿鲨嘎嘎猛吃，郭达千里单骑单挑巨鲨的视觉奇观，还能下饭。

观后感：下海炸大章鱼，上天插巨齿鲨。

海底奇观的表现较少，人鲨大战有点离谱，简直比举鼎的项羽还猛，不过作为电子榨菜还行吧，看这种电影，不能带脑子，评价也不能太严苛。巨齿鲨嘎嘎猛吃，让我想起了一款《饥饿鲨》的游戏，挺有意思的。

马亲王最近出的一本挺有意思的小书，虽是讲西游，但换个角度换个人物，简直就是一本天庭版的官场现形记、社畜职场人生指南，字字不提，字字诛心，读来有种身临其境的感受，彼时彼刻恰，如此时此刻。
